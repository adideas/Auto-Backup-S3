# Archiver

> Author: [Alexey Vlasov](https://github.com/adideas)

## API

| Method            | About                        |
|-------------------|------------------------------|
| archive(from, to) | Archive object (File/Folder) |

## USE
```java
public class Main {
    public static void main(String[] args) {
        Path from;
        Path to;
        
        Gzip.archive(from, to);
    }
}
```
